# 📚 SpringProject

## 프로젝트 목표
Spring 기초 학습 및 팀 프로젝트 진행. 기초적인 게시판 제작과 데이터 관리, 사용자 접근 등에 대해 배우기 위한 프로젝트 과정입니다. 다음과 같은 내용에 대해 학습합니다.
- Spring Framework와 Spring Boot의 기초 개념 이해
- CRUD 기능 구현을 통한 데이터 관리
- 사용자 인증 및 권한 관리
- 프로젝트 설계 및 팀 협업 방법론
